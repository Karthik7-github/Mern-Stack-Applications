# 📓 Diary Application

This is a simple **Diary Web Application** that allows users to store and retrieve daily entries. You can save information about a particular day and later search by date to view what happened on that specific day.

## 🚀 Features

* Add diary entries with a specific date
* Search entries by date
* View stored information for any selected day
* Simple and clean user interface

## 🛠️ Tech Stack

### Backend:

* Node.js
* Express.js
* MongoDB

### Frontend:

* HTML
* CSS

## ⚙️ How It Works

1. User enters details for a specific date
2. Data is sent to the backend via API
3. Backend stores the data in MongoDB
4. User can search by date
5. Matching entry is fetched and displayed

## ▶️ Setup Instructions

1. Clone the repository
2. Navigate to the backend folder
3. Install dependencies:

   ```
   npm install
   ```
4. Start the server:

   ```
   node server.js
   ```
5. Open the frontend (HTML file) in your browser

## 📌 Notes

* Make sure MongoDB is running locally or provide a connection string
* You may need to configure environment variables for database connection

## 💡 Use Case

This app is useful for keeping track of daily activities, notes, or memories and quickly retrieving them by date.

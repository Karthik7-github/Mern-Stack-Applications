This is a short project built on the MERN stack, using MongoDB for the database, HTML and CSS for the frontend, and Node.js with Express for the backend.

My project allows users to store daily activities securely and access them anytime to review what they’ve accomplished throughout the day.

My project is a daily activity management application built using the MERN stack, designed to help users record, store, and review what they accomplish each day. It allows users to securely save their daily tasks in MongoDB, while the frontend provides a simple and user-friendly interface using HTML and CSS. With Node.js and Express powering the backend, the system ensures smooth performance and reliable data handling. This project helps users stay organized, track their productivity, and easily reflect on their daily progress whenever they need.
